# Spring PetClinic DevOps Project

## Overview

This project demonstrates the containerization and deployment of the Spring PetClinic application using Docker and Docker Compose.

## Technologies Used

* Java 17
* Spring Boot
* Maven
* Docker
* Docker Compose
* MySQL
* Git
* GitHub

## Project Tasks

* Built the application using Maven
* Created a Dockerfile
* Built a Docker image
* Created Docker Compose configuration
* Deployed MySQL and application containers
* Managed source code using Git and GitHub

## Architecture

User
↓
PetClinic Application Container
↓
MySQL Container

Managed using Docker Compose
